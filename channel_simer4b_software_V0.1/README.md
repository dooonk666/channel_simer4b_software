# Channel Simulator SIMER4B

## 项目简介
Channel Simulator SIMER4B 是一个信道模拟器项目，专为无线通信系统的信道建模和仿真设计。它支持多径信道配置，能够模拟复杂的无线传播环境，最新版本支持 24 条多径配置，符合 3GPP 标准。

## 功能特点
- 支持从 JSON 文件导入信道配置
- 支持 12 条径扩展为 24 条径
- 支持多径时延、功率衰减、频移和瑞利衰落配置
- 提供图形化用户界面（GUI）
- 符合 3GPP 双极化信道模型标准
